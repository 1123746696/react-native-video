# react-native-video
适用于react-native的网络视频播放模块，适用于ios和android
# react-native-video
##安装
```
npm install react-native-video
rnpm link react-native-video
```

##API

| API | Note |    
|---|---|
| `play` | 播放视频 |



##Usage

```
import VideoPlayer from 'react-native-video'

//播放视频
VideoPlayer.play({url:'http://simple.mp4'});

```
